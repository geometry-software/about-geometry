Prototyping

Prototyping is a preparation phase for productive implementation. One goal of prototyping is to learn as many practical insights as possible while spending as little effort as possible. This can be achieved by quickly creating a simple prototype implementation, that you plan to throw away.

Firebase

Firebase is a set of backend cloud computing services and application development platforms provided by Google. It hosts databases, services, authentication, and integration for a variety of applications.

Sails

While frontend development focuses on designing and creating the user interface that users interact with, backend development operates behind the scenes, handling the server-side logic, security and database management that powers the entire system. Sails comes with blueprints that help jumpstart your app's backend with several lines of code and a powerful ORM, Waterline, which provides a simple data access layer that just works, no matter what database you're using.
